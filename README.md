# Cedra Link Flutter

Android application for Cedra Link.

## Tools

- Dart Version - 3.0.1
- Flutter Version - 3.10.1
- DevTools Version - 2.23.1

## Getting Started

```bash
# install dependencies
$ flutter pub get

# run on attached device
$ flutter run

# analyze code with flutter_lints
$ flutter analyze
```
